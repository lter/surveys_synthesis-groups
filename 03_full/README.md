# Wrangle & Visualize Full Working Group Survey Data

These scripts are needed to perform routine quality control operations on the full working group expectation surveys data and do desired visualizations. **These scripts require that the user has already run the "standardize" scripts and has those scripts' outputs in the proper local folders.** The "combo" scripts _also require that the quality control scripts for all three other full working group surveys have been run!_

Check the "purpose" comments in the top of each script for more information on what it does.

## Survey Timing / Name Clarification

See below for an explanation for the shorthand name of each full working group survey:

1. "early" - Administered at the end of the first in-person meeting
2. "mid" - Administered 1-1.5 years into a group's funding
3. "end" - Administered \~6 months after the final meeting of the group
4. "combo" - A synthetic combination of all questions that are found in more than one of the previous surveys
    - Useful for relating evolution of group thinking/satisfaction to group progress
