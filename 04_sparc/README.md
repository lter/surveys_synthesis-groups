# Wrangle & Visualize SPARC Survey Data

These scripts are needed to perform routine quality control operations on the Scientific Peers Accelerating Research Collaborations (SPARC) expectations survey data (typically administered \~6 months after their in-person meeting) and do desired visualizations. **These scripts require that the user has already run the "standardize" scripts and has those scripts' outputs in the proper local folders.**

Check the "purpose" comments in the top of each script for more information on what it does.
